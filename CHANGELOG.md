## [0.1.0] - 2025-11-13

### Added
- Initial release of n8n-nodes-outplay
- Outplay node with prospect operations:
  - Save Prospect (create/update)
  - Get Prospect by ID
- Outplay Trigger node for webhook events
- API Key authentication with location-based endpoints
- Support for custom Outplay branding with outplay-icon.svg
